FREQUENTLY ASKED QUESTIONS

How do I add GitHub stats cards to my profile?

Simply enter your GitHub username, select a theme, click "Execute" to preview, then click "Copy MD" to copy the markdown code. Paste it into your GitHub profile README.md file and commit the changes.

Do I need a GitHub token or authentication?

No! You don't need to sign up, authenticate, or provide any tokens. Just enter your GitHub username and the tool handles everything automatically.

How often do the stats update?

Stats are cached for 1 hour for optimal performance. After 1 hour, the cards automatically fetch fresh data from GitHub's API when viewed.

Can I use different themes for stats and languages cards?

Yes! You can mix and match themes. For example, use Cyberpunk for your stats card and Luxury for your languages card to create a unique profile look.

Is this tool free to use?

Yes, completely free! This is an open-source project under MIT License. You can use it for personal or commercial purposes without any restrictions.

What makes this different from github-readme-stats?

While github-readme-stats is excellent, our tool focuses on unique, artistic themes with completely different visual identities. Each theme (Cyberpunk, Brutalist, Terminal, Luxury, Vaporwave) offers a distinct aesthetic that stands out from traditional stat cards.
